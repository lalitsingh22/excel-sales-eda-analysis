# 🍪 Cookies Sales Data Analysis – Exploratory Data Analysis (EDA) using Excel

## 1. 📊 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a cookies sales dataset to uncover meaningful business insights and sales patterns. The analysis was conducted using **Microsoft Excel**, including data cleaning, transformation, and visualization.

The objective of this project is to transform raw sales data into actionable insights through structured analysis and an interactive dashboard.

This project demonstrates key **data analyst skills** such as:

* Data cleaning
* Data transformation
* Exploratory analysis
* Data visualization
* Business insight generation

---

# 2. 🎯 Objective of the Analysis

The main objectives of this project are:

* Analyze overall **sales performance**
* Identify **top-performing cookie products**
* Understand **regional sales distribution**
* Discover **sales trends over time**
* Build a **visual dashboard in Excel** for better decision-making
* Extract **business insights from the dataset**

---

# 3. 📁 Dataset Description

### Raw Dataset

The raw dataset contains the **original transactional sales data** before any preprocessing.
It includes uncleaned entries such as possible duplicates, inconsistent formatting, and missing values.

Typical fields include:

* Order Date
* Region
* Product (Cookie Type)
* Units Sold
* Revenue / Sales Amount

### Cleaned Dataset

The cleaned dataset is the **processed version of the raw dataset**, prepared for analysis.

Cleaning steps were performed to:

* Remove inconsistencies
* Format data correctly
* Prepare the dataset for pivot tables and visual analysis

This dataset was used to build the **EDA and Excel dashboard**.

---

# 4. 🧹 Data Cleaning Process

The following data preprocessing steps were applied:

### Handling Missing Values

* Checked for missing entries in important fields
* Handled or removed incomplete records where necessary

### Removing Duplicates

* Duplicate records were identified and removed to ensure accurate analysis

### Data Formatting

* Standardized **date formats**
* Ensured numeric columns (sales, units sold) were properly formatted
* Corrected text inconsistencies

### Data Transformation

* Organized columns for analysis
* Prepared dataset for **Pivot Tables and Charts**
* Structured the dataset for dashboard creation
